# 📊 COVID-19 Data Analysis Dashboard using Power BI

## 📌 Project Overview
This project presents an **interactive COVID-19 Data Analysis Dashboard** created using **Microsoft Power BI**.  
The dashboard provides a clear and visual understanding of COVID-19 case distribution across Indian states, helping users explore trends, comparisons, and key metrics effectively.

The goal of this project is to transform raw COVID-19 data into **meaningful insights** through data cleaning, modeling, and visualization.

---

## 🎯 Objectives
- Analyze COVID-19 data at a **state-wise level**
- Track **Active, Death, and Discharged cases**
- Compare **total cases and population distribution**
- Enable **interactive exploration** using filters and slicers
- Present insights in a **clear and user-friendly dashboard**

---

## 📊 Dashboard Features
- **KPI Cards** showing:
  - Active Cases
  - Death Cases
  - Discharged Cases
- **State-wise Total Cases Analysis**
- **Population vs Total Cases Visualization**
- **Interactive Slicers** for state selection
- Clean, minimal, and intuitive dashboard design

---

## 🛠 Tech Stack
- **Microsoft Power BI Desktop** Main data visualization platform used for Report Creation
- **Power Query** for data cleaning and transformation
- **DAX** (Data Analysis Expression) for calculated measures and KPIs
- **Data Modeling** Relationships establied among tables
- **Data Visualization**

---

## 📂 Dataset
- COVID-19 dataset  
- Contains information on:
  - State-wise COVID-19 cases
  - Active, Death, and Discharged counts
  - Population data
